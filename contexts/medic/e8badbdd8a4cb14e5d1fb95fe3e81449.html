Publisher's Description

Improve your PC performance by repairing registry errors: The Windows registry is a huge database that ensures normal computer operation. Installing and uninstalling software can make your registry a mess, leading to decreased PC performance and causing computer crashes. Registry Medic uses a highly intelligent engine to scan your computer's registry for invalid registry entries.

Registry Medic is software which can repair the Windows registry.

Registry Medic features and highlights:
  • Scan system extensions and windows firewall applications.
  • Registry Firewall monitors and blocks malicious software on your computer.
