Re: Computer guys: Difference between system image disc, etc
In addition to using an image file to restore your computer you can also mount the image on a different computer and view/edit/restore selected files.

Also, if you are using image files to backup your computer you should do a test recovery to ensure that the image is readable. I recently had an image fail when I attempted to restore it to the computer, even though I could mount and view the image/ files on a different computer.

Re: Computer guys: Difference between system image disc, etc
The disk imaging included in Windows 7 works beautifully. You will also need to create the system repair disc to use it correctly. In case of a disk failure all you have to do is install a new drive and boot from the repair disc. Once booted you will have the chance to restore your system from an existing image. Works like a charm.

Of course you should create your image on a separate drive, either external or internal.
